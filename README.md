<h1 align="center">Lens Locked</h1>

An awesome photo gallery application written in Go!

## Instructions

### Run in development

```bash
go run *.go
```

### Run in production

```bash
go run *.go -prod
```

### Getting help

```bash
go run *.go --help
```

### Build the app

```bash
go build -o app *.go
```
